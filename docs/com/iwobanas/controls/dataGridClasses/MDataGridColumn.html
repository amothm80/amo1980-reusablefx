Classpublic class MDataGridColumn
InheritanceMDataGridColumn mx.controls.dataGridClasses.DataGridColumn

The MDataGridColumn class describes a column in a MDataGrid control. It extends DataGridColumn functionality by adding filtering support. MDataGridColumn also changes default renderers. Default item renderer is BoldSearchItemRenderer and default header renderer is DropDownFilterHeaderRenderer.

filterEditorproperty
public var filterEditor:IFactory

The class factory for filter editor instances that edits column filter values. Objects created by this factory should implement IColumnFilterEditor interface.

The default value is WildcardFilterEditor.
